介绍

Uzi是新一代云平台即服务（PaaS），支持形形色色的编程语言。 借助我们的平台，无论是大规模的企业API，还是个人的静态网站，你的网络应用都可以即时部署，即时上线。现在填写表格，即可参与试用。

如果您拥有一定的HTML，CSS与JavaScript基础，那么你就可以轻易地上手我们的服务！

快速上手

最新稳定版本为21.03，测试版本为21.06(alpha)。点击下方按钮进入下载页面，或者查看历史版本。

最新稳定版本 历史版本

环境要求

  • 支持 Windows 10, Mac OSX 或者 Linux
  • Node.js v12及更高版本

安装

Mac (Homebrew)

$ brew install Uzi-cli

NPM

$ npm install Uzi-cli

Yarn

$ yarn install Uzi-cli
